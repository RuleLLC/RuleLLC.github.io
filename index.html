Kest

A clean operating system

Kest is an operating system based on the Linux kernel (a Linux distribution) that keeps things simple when it comes to the internal structure of the system. When it comes to security, you can be assured that there are no memory exploits, because almost all code is written in Rust. Here are the main features of Kest:

  • Embedded desktop system
  • Fast disk encryption
  • App sandboxing
  • Customizable system
